# Default Focus — Initial

## Overview

When the Amend Request screen first opens, the keyboard focus is placed on the **Request No.** field. This ensures that registration staff can immediately type a request number without needing to click the field manually, streamlining the retrieval workflow from the moment the screen appears.

---

## Related User Stories

- **[[CRST-785]]** - Amend Request - Default Focus (Initial)

---

## Behaviour

When the Amend Request screen is opened — for example, by selecting **Amend Request** from the Registration top menu — the system places the initial keyboard focus on the **Request No.** field.

The **Request No.** field is visible and editable from the moment the screen opens, so staff can begin entering a request number immediately.
